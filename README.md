# Bitmap Punker
![](https://github.com/Barclay-McClay/BitmapPunker/blob/main/demo.gif)
Requires

Google Chrome installed on Windows PC with Python 3

```
pip install selenium
pip install pillow
```

```
"Inscription ID",
"Block Number",
"Date",
"Value",
"Bytes",
"Transactions",
"Sub K Range",
"Negative Punk",
"Historical Punk",
"Perfect Punk",
"Bitfeed Link",
"Unisat Link"
```
